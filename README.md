# SOCKS5 Proxy Manager 🚀

Интерактивный менеджер для создания и управления SOCKS5-прокси на базе Dante Server (Ubuntu/Debian).

## 🌟 Основные возможности

- **Множественные прокси**: Создавайте сколько угодно пользователей на разных портах.
- **Автоматизация**: Скрипт сам генерирует порты, логины и пароли (или позволяет вводить их вручную).
- **Интерактивное меню**: Удобное управление (Добавление, Список, Удаление, Статус).
- **Ссылки для Telegram**: Автоматическая генерация ссылок форматов `tg://socks?` и `https://t.me/socks?`.
- **Безопасность**:
  - Создание системных пользователей без доступа к shell (`/bin/false`).
  - Автоматическая настройка Firewall (ufw).
  - Изоляция конфигурации для каждого прокси.

## 🚀 Быстрый запуск

Выполните команду на вашем сервере (от имени root):

```bash
wget -qO sock5-manager.sh https://raw.githubusercontent.com/qwerokip-wq/sock5-manager/main/sock5-manager.sh && chmod +x sock5-manager.sh && sudo ./sock5-manager.sh
```

## 🛠 Ручная установка

1. **Клонируйте репозиторий или скачайте скрипт:**
   ```bash
   git clone https://github.com/qwerokip-wq/sock5-manager.git
   cd sock5-manager
   ```

2. **Сделайте скрипт исполняемым:**
   ```bash
   chmod +x sock5-manager.sh
   ```

3. **Запустите менеджер:**
   ```bash
   sudo ./sock5-manager.sh
   ```

## 📖 Меню управления

После запуска вам будут доступны следующие опции:
1. **Добавить новый прокси** — создание нового пользователя и порта.
2. **Список прокси** — просмотр всех активных подключений и ссылок.
3. **Удалить прокси** — выбор прокси по номеру из списка для удаления.
4. **Проверить состояние** — диагностика службы Dante и проверка открытых портов.
5. **Удалить менеджер** — полная очистка сервера от прокси и настроек.

## 📋 Требования
- ОС: Ubuntu 20.04+ / Debian 10+
- Права: Root доступ
